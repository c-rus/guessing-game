# Guessing Game

## The guessing game program implemented from [The Rust Programming Language](https://doc.rust-lang.org/book/ch02-00-guessing-game-tutorial.html#programming-a-guessing-game) book.

### Overview

This project was created to gain an initial understanding of rust, and experiment with DevOps for a building a rust application.

### How to play

- Enter a number ranging from 1-256.

- Continue guessing a number until you've guessed correctly!
